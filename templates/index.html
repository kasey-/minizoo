Inception

The inception neural network is one of the most famous classifiers. Hereafter I am putting into demo the version 3 trained on the famous Imagenet database making it able to recognize 10 000 types of objects on a picture. Even if the implementation of it was pure thanks to modern machine learning framework, this application is a great first step before YOLO algorithm.

MNIST

The MNIST database is a set of 70,000 handwritten digits and pre-classified making it a perfect base for a classification exercise. All samples are in black and white and pre-normalized in size and characters position. The MNIST is a subset of the more extensive NIST database.

Prophet

Facebok prophet is a machine learning tool specialized in forecasting future data based on historical observation for time series. Prophet interagrate a set of tool making it robust to data missing, it integrate seasonality and is very able to auto-tune itself to produice the best result.

Titanic

The Titanic incident is one of the most famous in the history. On April 15, 1912, during its inauguration trip, the Titanic sank after a collision with an iceberg. From this incident, only 1502 of the 2224 passengers and crew survived. Thanks to the availability of the data, the prediction of the survival of the Titanic accident is a well-known machine learning exercise.
